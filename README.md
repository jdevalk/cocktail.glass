# cocktail.glass

An Astro site for browsing cocktail recipes by category, glass, and ingredient.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Static output is generated in `dist/`.

## Cloudflare Pages

This project is set up for Cloudflare Pages with:

- Build command: `npm run build`
- Build output directory: `dist`

You can deploy either by:

1. Connecting this GitHub repo to Cloudflare Pages.
2. Using Wrangler directly:

```bash
npm run deploy:pages
```
